# oo0-STATE

Mind (Agent Zero), Heart (OpenClaw), and Conscience (arifOS)

## Repository Structure

This repository contains the following integrated projects:

- **[agent-zero/](agent-zero/)** - AGI: A personal, organic agentic framework that grows and learns with you
  - Mind component of the oo0-STATE system
  - Source: https://github.com/agent0ai/agent-zero

- **[openclaw/](openclaw/)** - ASI: Personal AI Assistant that runs on your own devices
  - Heart component of the oo0-STATE system
  - Source: https://github.com/openclaw/openclaw

- **[arifOS/](arifOS/)** - Constitutional AI Governance System
  - Conscience component of the oo0-STATE system
  - Source: https://github.com/ariffazil/arifOS

- **[opencode/](opencode/)** - APEX: The open source AI coding agent
  - Source: https://github.com/anomalyco/opencode

Each subdirectory contains a complete, standalone project. Please refer to the individual README files in each directory for specific documentation and installation instructions.
