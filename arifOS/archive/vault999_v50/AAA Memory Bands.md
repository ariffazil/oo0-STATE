## Memory Band Structure

### **L0: Genesis (Constitutional Foundation)**
- Core identity and values
- Immutable principles
- Constitutional oath

### **L1: Identity (Self-Concept)**
- Role definition
- Capabilities and boundaries
- Agent personality

### **L2: Knowledge (Learned Facts)**
- Domain knowledge
- Patterns and insights
- Learned truths

### **L3: Episodic (Event Memories)**
- Session histories
- Significant events
- Temporal context

### **L4: Procedural (How-To Knowledge)**
- Workflows and processes
- Best practices
- Skill knowledge

### **L5: Void (Archived/Forgotten)**
- Deprecated knowledge
- Historical context
- Archived memories

---

## How Memory Bands Work

Memory flows DOWN through the bands (L0→L5):
- **L0** is strongest (constitutional foundation)
- **L5** is weakest (archived/forgotten)

**Principle:** L0 overrides all other bands (constitutional supremacy)
