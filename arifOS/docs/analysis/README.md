## Usage

### Running Analysis

1. Open GitHub Copilot Chat in VS Code (`Ctrl+Shift+I`)
2. Paste prompts from `.github/COPILOT_ANALYSIS_PROMPTS.md`
3. Save outputs to files in this directory
4. Consolidate into `CONSOLIDATED_ANALYSIS.md`

### Creating Consolidated Report

```bash
cd docs/analysis/
cat 01_*.md 02_*.md 03_*.md 04_*.md 05_*.md > CONSOLIDATED_ANALYSIS.md
```

### Tracking Metrics

Create `metrics/baseline_metrics.md`:

```markdown
# arifOS Optimization Metrics

```

## Analysis Schedule

- **Monthly:** Run Prompt 1 (Architecture Scan) to track improvements
- **Quarterly:** Run all 5 prompts for comprehensive review
- **After major releases:** Full analysis to validate changes

## Tips

### Efficient Analysis

Run prompts in parallel:

```bash
# Terminal 1: Prompt 1
code --goto docs/analysis/01_architecture_scan.md

# Terminal 2: Prompt 2
code --goto docs/analysis/02_action_plan.md

# etc.
```

### Version Control

Track analysis evolution:

```bash
git log --oneline docs/analysis/01_architecture_scan.md
```

### Diff Between Analyses

Compare improvements:

```bash
git diff HEAD~1 HEAD -- docs/analysis/01_architecture_scan.md
```
